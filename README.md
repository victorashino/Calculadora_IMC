## Tecnologias
* Kotlin
* Activity para host de telas
* ViewModel
* ViewGroup
    * ConstraintLayout
    * LinearLayout
- Components de UI
    - TextView
    - EditText
    - Button
- Toast para display de mensagens
- Intent para abrir a segunda Activity
- viewBinding para recuperar components de UI
- Animções para transição de telas
- Room
- MVVM
- Mockito
- Testes
